# Vue 3 + TypeScript + Vite

```
工程创建指令
npm create vite@latest yolo-helmet --template vue-ts
```

基于 YOLO 的工程施工员安全帽检测项目
# 技术栈
## 前端技术栈

    HTML5/CSS3：页面结构和样式
    Vue.js 3：前端响应式框架
    Element Plus 2.4.1：UI 组件库
    JavaScript：前端交互逻辑
    Echarts: 前端图表库（可视化检测统计数据）

## 算法端技术栈

    Python 3：主要编程语言
    Flask 2.3.3：Web 应用框架
    Flask-CORS 4.0.0：跨域资源共享支持
    Ultralytics 8.0.200：YOLO 模型库（基于 YOLO11n 实现安全帽检测）
    PyTorch 2.0+：深度学习框架
    Pillow 10.0.1：图像处理库（图片预处理、格式转换）

# 功能模块
## 用户管理

    用户登录：支持用户名密码登录
    用户注册：新用户可注册账号
    账户验证：确保系统安全性（仅授权用户可使用检测功能）

## 安全帽检测
    图像上传：支持本地施工场景图片 / 视频帧文件上传
    实时检测：基于 YOLO11n 模型进行施工员安全帽佩戴状态检测（分类：佩戴安全帽、未佩戴安全帽）
    结果展示：显示检测结果及置信度，支持单张图片多目标检测结果标注，展示 Top3 预测结果（针对疑似目标）
    多模型支持：提供已训练通用安全帽检测模型和定制化训练模型两种选择

## 模型数据查看
    训练数据展示：查看模型训练过程中的损失曲线、迭代次数等核心数据和指标
    验证结果展示：查看模型在测试集上的漏检率、误检率、精准率等性能表现
    精度统计：展示模型检测精度（mAP）、召回率、F1 值等关键指标
